# KanbanBoard Component `kanban-board` This element represent a Kanban board where various columns can be defined. ## Simple Example In this example we have a kanban board with 3 columns, defined by the `kanban-column` element, the tasks can be any element, native or other web component. The user will be able to drag and drop tasks between columns. _Note:_ _the `title` attribute on the board and columns will be used on the UI, so make it human readable._
